About the project

This project involves building a personal website hosted on AWS, implementing a CI/CD pipeline, and deploying the website with a back-end API. The website, hosted on Amazon S3 and delivered via CloudFront, is secured with HTTPS for encrypted communication. For the backend, an AWS Lambda function, integrated with API Gateway, handles visitor counting logic, and a DynamoDB table stores the data securely.
